개인프로젝트 인스타 ui 활용한 인터넷 게시판 기능 구현

프로젝트 작업 기간: 한 달
인원 : 유나미
사용한 기술 툴 JPA springboot aws putty winScp Thymeleaf

•파일 업로드 가능한 게시판 기능
•NativeQuery로 지난 한 시간 동안 가장 많은 좋아요를 받은 게시글의 타이틀을 추출
•검색할 때 쿼리문을 사용해 "가"를 입력해도 가로 시작하는 게시글과 타이틀 나오게하기
•비밀번호 잊어버렸을 때 MimeMessage 사용해 이메일로 변경된 비밀번호 유저한테 보내기
•Putty로 EC2 인스턴스에 접속한 후, 스프링부트를 설치
•WinSCP를 활용해 JAR 파일을 서버에 업로드한 다음, EC2에서 실행시켜 애플리케이션을 구동

작업 중 힘들었던 점-> 파일 업로드 과정에서 Windows와 Ubuntu 간의 파일 처리 방식 차이로 인해 상당한 어려움을 겪었습니다. 이를 해결하기 위해 Ubuntu 리눅스에 대한 심층적인 학습을 진행하였으며, 이 과정에서 SSH와 TCP와 같은 네트워크 프로토콜이 데이터 전송에 미치는 영향을 이해하게 되었습니다. 또한, 도메인 및 인증서 관리에 대한 지식을 쌓으면서 리눅스 명령어와 시스템 관리 기술을 익혔습니다. EC2 인스턴스를 활용하여 IP 주소를 설정하고 작업하는 과정에서 발생한 문제를 해결하기 위해 CloudWatch를 도입하여 경고 시스템을 구성하였고, 이를 통해 CPU 사용량을 보다 정밀하게 모니터링할 수 있었습니다.
